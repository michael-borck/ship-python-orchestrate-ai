## Scaffold Script

The bash scaffold script from Chapter 1 creates a basic project structure:

```bash
curl -O https://raw.githubusercontent.com/michael-borck/ship-python-template/main/scripts/scaffold_python_project.sh
chmod +x scaffold_python_project.sh
./scaffold_python_project.sh my_project
```

## Notes

This book focuses on development infrastructure rather than standalone code examples. The companion repositories above contain complete, working implementations of the practices described in each chapter.
